# PoCX Address Tracker

A modern Android app to track PoCX addresses, balances, blocks and transactions.

## Features

- Track multiple PoCX addresses
- Push notifications for won blocks
- Multi-language support (DE/EN)
- Explorer API configurable
- TX_ID clickable
- Stats dashboard
- Address labels
- Pull-to-refresh
- Dark blue UI design

## API

Uses:
https://explorer.bitcoin-pocx.org/api/
